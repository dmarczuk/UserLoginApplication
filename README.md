# User Login Application

## About project
Simple REST application, allowing register and login user. Also user can update your data and remove users from database.


## Main responsibilities application
<ul>
<li>Login User</li>
<li>Register User</li>
<li>Update user</li>
<li>Show users</li>
<li>Delete user</li>
</ul>
